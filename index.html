Cara Saya Mendapatkan Rp1.000.000.000.000 Dalam Satu Hari

Cara Saya Mendapatkan Rp1.000.000.000.000 Dalam Satu Hari

Selasa, 31 Maret 2020

Ingin mendapatkan banyak uang dengan cepat?

Ingin mendapatkan banyak uang dengan cepat? Pernah bermimpi mendapatkan semua yang kalian inginkan sekaligus? Jika “Ya” teruskanlah membaca artikel ini karena saya akan memberi tahu anda bagaimana caranya. Dengan membaca artikel ini, kalian akan tahu bagaimana cara untuk keluar dari pekerjaan membosankan anda dan menghasilkan uang Rp1.000.000.000.000 dalam sekejap. Saya sudah berhasil mendapatkannya, dan jika kalian mau, kalian pun bisa. Saya tidak merasa rugi karena harus berbagi rahasia ini dengan anda. Justru, rahasia ini akan membantu kalian yang ingin mengubah hidup menjadi lebih baik dan bebas secara finansial.

Introduce Myself

Pertama-tama, perkenalkan nama saya Agung Hartanto. Usia saya 27 tahun dan tinggal di Jakarta. Saya hanyalah seorang laki-laki biasa yang terlahir di keluarga sederhana atau hampir kurang mampu. Saya terlahir dari orang tua yang sudah paruh baya. Pekerjaan ayah saya hanyalah kuli bangunan dengan upah harian Rp50.000 perhari. Sedangkan ibu saya hanya seorang asisten rumah tangga. Seingat saya, sedari kecil orang tua saya sudah bekerja keras untuk membiayai sekolah saya.
Untuk mencukupi kebutuhan sehari-hari saja terkadang masih kurang. Apalagi untuk membeli pakaian. Kalau cukup beruntung, terkadang majikan ibu saya memberikan baju untuk kami pakai sehari-hari. Ketika saya lulus SMA, saya tidak pernah berfikir untuk kuliah karena saya harus mencari uang untuk orang tua yang tidak lagi muda. Akhirnya, saya memutuskan untuk bekerja di sebuah café di daerah perkotaan dengan gaji perbulan Rp2.000.000. Saat itu, 2014, gaji tersebut sudah terbilang besar di Indonesia. Saya menghasilkan uang yang cukup untuk memenuhi kebutuhan sehari-hari, tetapi saya memiliki mimpi untuk bisa membeli rumah baru untuk orang tua saya. Saya tahu, rumah itu sangatlah mahal dan saya harus menabung bertahun-tahun untuk membelinya. Tetapi tidak apa. Bagaimanapun juga, itulah impian terbesar saya dan saya tahu saya tidak akan bisa mewujudkannya dalam 1 malam. Setidaknya, itulah yang ada di benak saya dulu. Saya sangat tertekan dengan kondisi keuangan saya saat itu, harga terus naik karena inflasi dan hidup di Jakarta amatlah keras. Orang-orang seperti saya galau, tetapi saya harus tetap bekerja.
Namun, setelah beberapa bulan berlalu cafe tempat saya bekerja bangkrut lalu tutup begitu saja. Kehidupan saya semakin buruk tanpa pekerjaan dan smber pemasukan. Saya hanya hidup dari hasil tabungan orang tua saya yang sangat sedikit. Itu adalah masa-masa terberat yang saya alami. Mati-matian saya mencoba mencari lowongan di Internet, namun karena hanya bermodalkan ijazah SMA, hasilnya nihil.

Kesempatan Ada Di Depan Mata Saya

Dua minggu kemudian, saya mendapatkan sebuah halaman web yang mengisahkan RP200.000.000 dari sebuah lotere dan ia juga mencantumkan situs lotere tersebut. Saya berfikir, mungkin inilah peluang saya untuk mendapatkan uang banyak dalam waktu dekat. Dan mungkin kesempatan ini hanya ada sekali dalam seumur hidup. Awalnya, saya benar-benar kesulitan dalam menentukan nomor yang akan saya beli. Tetapi kemudian saya mencari tips dan trik permainan tersebut. banyak informasi yang saya dapatkan melalui website, blog, forum, dan lain-lain. Dari sumber tersebut, akhirnya saya mencoba untuk mempraktikkannya. Saya sangat bahagia hanya karena bisa mengerti cara menentukan angka tersebut. Ketika saya akhirnya benar-benar paham prosesnya, akhirnya saya mencoba untuk membelinya pertama kali. Walaupun hanya bermodalkan uang tabungan sisa saya bekerja di cafe dulu. Saya mendaftar akun lotere tersebut dan mendapatkan tata cara untuk membeli undian. Lalu, saya mulai membeli 6 angka vertikal dengan angka yang bervariasi. Saya melakukan pembelian hari itu juga lalu menunggu pengocokkan undiannya. Tidak disangka, saya langsung mendapatkan hadiahnya! Saat itu, hadiahnya baru berjumlah Rp10.000.000.
Namun, itu sudah termasuk angka yang besar. Apalagi dengan mengingat bahwa saya tidak melakukan apapun. Hanya duduk didepan TV dan Komputer. Hari selanjutnya, saya membeli dan menentukan lagi angka yang akan saya pasang selanjutnya. Seharian penuh saya hanya tiduran melihat HP. Saya sampai bergadang demi mendapatkan nomor yang tepat. Saat itu sudah jam 3 subuh dan akhirnya saya mendapatkan nomor yang akan saya pasang. Ingin rasanya saya langsung keluar membelinya. Namun, saat itu perdagangan masih tutup dan saya perlu istirahat. Akhirnya saya merebahkan badan di kasur dan terlelap. Esoknya, saya bangun jam 1 siang dan langsung membuka HP saya kembali. Saldo yang saya menangkan masih tersedia dan siap untuk di cairkan. Jadi, ini bukan mimpi! Saya lalu membeli angka yang telah saya pilih tersebut lalu memasangnya. Sambil menunggu pengocokkan selanjutnya, saya bergegas pergi ke bank untuk membuka rekening pertama saya. Setelah itu, saya pulang dan mendaftarkan nomor rekening saya ke akun lotere tersebut. Lalu, dua jam kemudia saya mendapatkan pesan dari bank bahwa rekening saya baru saja dikreditkan! Saya benar-benar kaget sekaligus bahagia. Dan saat saya cek, memang benar saldo tersebut telah masuk! Minggu berikutnya, saya terus bermain lotere dan terus-terusan menang. Saya menghasilkan lebih dari Rp500.000.000 dalam seminggu. Begitu terus sampai akhirnya saya bisa membeli rumah pertama saya. Impian utama saya yang hampir saya lupakan. Lalu pada suatu hari, saya kembali memasang angka untuk lotere, dan kalian harus tahu apa yang terjadi. Saya mendapatkan langsung JACKPOT sebesar Rp1.000.000.000. Angka tersebut merupakan angka terbesar dalam sejarah lotere mereka. Sekarang, sambil mengetik tulisan ini, saya sambil mengingat-ngingat betapa senang dan betapa bahagianya saya serta keluarga dulu. Bagaimana bisa? Saya hanya yakin pada diri saya sendiri! Awalnya saya hanya menemukan peluang dan kemudian saya yakin bahwa saya juga bisa menghasilkan uang seperti orang yang ada di halaman web tersebut. Dan hal itu memang benar terjadi!

Jadi, Anda Harus Yakin Bahwa Anda Juga Bisa

Jadi, anda harus yakin bahwa anda juga bisa. Keyakinan itu sangat penting karena anda tidak akan bisa meraih apa pun tanpa keyakinan. Tidak akan pernah! Jika anda tidak percaya pada diri sendiri, anda akan mulai menganggap bahwa semua peluang terbaik dalam hidup anda sudah hilang. Dan itulah yang akan terjadi jika selalu berfikir demikian. Jika anda putus asa dan tidak mempercayai apapun atau siapapun, anda tidak akan dapat melihat peluang yang mengubah hidup anda. Cobalah untuk melihat peluang dimanapun, di setiap kesempatan yang terjadi di sekitar anda, di setiap orang yang anda temui, dan di setiap kisah yang anda baca. Dimana saja! Selalu cobalah untuk memanfaatkan peluang tersebut karena peluang tersebut benar-benar milik anda! Dan anda akan meraihnya! Karena beberapa alasan, saya selalu percaya bahwa saya akan bisa membeli rumah dan membahagiakan orang tua saya suatu hari nanti. Dan saya benar-benar mewujudkannya! Saya tidak perlu menabung dan langsung membelinya karena saya memiliki uang yang cukup untuk membayar. Saya memiliki sumber pendapatan yang tidak akan pernah habis. Bahkan, semakin banyak setiap harinya. Dan, saya juga tidak perlu lagi untuk melapor untuk bekerja berjam-jam lamanya tanpa ada kesempatan untuk duduk seperti yang saya alami sebelumnya. Saya sekarang dalam sehari dapat menghasilkan uang lebih banyak dibandingkan dengan yang dulu saya hasilkan dalam sebulan. Dan saya benar-benar merasa bebas!! Apakah itu benar-benar suatu keajaiban? Bisa jadi. Tapi sekali lagi, itu bukan mimpi. Dan sangat nyata. Saya hanya membaca sebuah artikel lalu mengambil keputusan. Itulah cara saya mencapainya. Oh ya, saya rasa itu saja dulu, karena sudah terlalu jauh dari topik.
